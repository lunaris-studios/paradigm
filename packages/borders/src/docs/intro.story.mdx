The [**@paradigmjs/borders**](https://www.npmjs.com/package/@paradigmjs/borders) provides several components for creating high fidelity, scalable, and interactive borders.

- [NineSlice](/?path=/story/borders-nineslice--basic) for creating high-fidelity, scalable borders. [You can read more on 9-slice scaling here](https://en.wikipedia.org/wiki/9-slice_scaling).

### Installing Borders

Borders is currently under heavy development, but can be installed by running either of these commands:

```
npm install @paradigmjs/borders
```
or
```
yarn add @paradigmjs/borders
```
